pldrGallery

pldrGallery is web application for file browsing that focuses on viewing images and videos on custom private servers. Originaly it started just as a simple PHP script, but as time was flowing, I was looking for a more robust solution and pldrGallery was born!

The basic goal is to not give gigabytes of your personal media files and pictures from holidays to public servers, but keep them on your own computer, while keeping the ability to share them with anyone out in the Internet.

All gallery projects that I found were creating thumbnails and indexes, didn't care about security or simply didnt't work with so much collected data.
When I started this project, my family gallery was about 500 GB big with ~80 000 files and ~1200 folders and I knew it will be growing.

Features

  • Fast browsing of folders and files on your hard drive without any need to reload the page
  • Image, video and audio is fully supported thanks to HML5. Other files can be downloaded directly
  • Support for Google login and shareable links (passwords)
  • Even if you know URL to a file or a folder, you will not access it unless you have permission.
  • Permissions are set individually for each folder or file, and can be customized as needed
  • Data saver to run even on slow internet connection (server upload or client download)
  • Relax with presentation mode
  • Show images on map based on EXIF data
  • Save your eyes with dark theme
  • Use keyboard for almost everything
  • And much more...
