Permission

For me is crucial to have shared only some of folders. But some closest members should have access to everything. Also I want to share some folders to social network or chat, but without knowing who is in the chat.

Security

All folders and files are visible and accessible ONLY if you have permission for them. That means, even if you know URL, you cannot access them, don't matter if it is folder, image, audio, header etc.
You can test it in this playground. For example, this is direct link to one of files in this folder. This will work ONLY if you have master password (more info below).

User permissions

Do you want share some folder only to your wife? Easy, just look on the examples in .pmg_perms in root folder.
Sorry, there is no way how to show live demo of this :( You have to install and test yourself.

Passwords

Items can be also shared via passwords, just add some in .pmg_passwords in root folder. There are some examples below how to see more items in the structure.
If is password valid, you will be redirected to that folder automatically.

  1. As you probably noticed, there is no footer. If you want to see it, just use password "permissions-show-footer"
  2. Also, there are no folders in here... But let see some of them with password "more permissions!". This will also redirect you into one of these folders.
  3. Lets see some files with password "secured images". Permissions in this case are defined with prefix.
    Btw. did you noticed, that there is space in this password? In previous password there is exclamation mark. Only ; and , characters are not allowed as password.
  4. Password "aaaaaaaa" don't exists.
  5. To access every file with master password, use "access-to-everything".


To see saved password, just visit bottom section of settings (or see JSON formatted here)

To clear saved passwords, just clear your cookies (specifically pmg-passwords).
