Demo header

If you want to have this "greeting" text on the top of a loaded structure, just place a header.html file in the folder and it will get automatically rendered into div#structure-header.
You can use any tags, entities, javascipt, csss, whatever, it is just up to you if you break your own page ;)
It works exactly the same for footer.html, which is rendered under the loaded structure into div#structure-footer.
