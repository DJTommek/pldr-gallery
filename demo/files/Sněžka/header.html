Data saving

There are also pretty big files in this folder so it is great to test "Data saving" feature in the settings.
Basically it just resize image to maximum dimensions 1024 x 1024 (by default). If you want to show file in original size even with "Data saving" feature enabled, just open file into popup and click on button.
