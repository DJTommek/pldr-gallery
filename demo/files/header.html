Presentation

Presentation mode can be started or stopped via button in popup, which appears when a file is opened. It will automatically go to the next file, and the behavior depends on the file type:
Image: after the image is loaded, gallery will wait 5 seconds (adjustable in settings) and then go to the next file
Video: after the video is loaded, it will be played. As soon as the video stops, gallery will go to the next file.
Audio: behaves just like video
Any other file: still in @TODO

After displaying the last file, presentation mode will stop.
