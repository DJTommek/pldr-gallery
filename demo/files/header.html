In this folder you can see mixed types of files to browse.

Presentation

Presentation mode can be started or stopped via button in popup. It will automatically go to next file and depends on the file type what happens next:
Image: after image full load, it will wait 5 seconds (adjustable in settings) and then go to next file
Video: after video load, it will start playing the video. As soon as video will stop, it will go to next file.
Audio: the same as video
Any other file: still in @TODO

If it hits end, presentation mode will stop.
