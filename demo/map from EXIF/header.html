Trip to the highest mountain in Czechia, Sněžka in the Krkonoše mountains + one extra video file to demonstrate, that location can be loaded from videos too.

EXIF and map

Here you can test the map generated from EXIF info loaded from every file in this folder.

Data saving

There are also pretty big files in this folder, so it is a good opportunity to test the "Data saving" feature from the settings.
Basically, it just scales the image to be smaller than 1024px in both dimensions (by default). If you want to show the original file even with "Data saving" enabled, just open it and click the button in the popup.
